apk打包流程

  1. 打包资源文件，生成R.java文件
  2. 处理AIDL文件，生成对应的.java文件
  3. 编译Java文件，生成对应的.class文件
  4. 把.class文件转化成Davik VM支持的.dex文件
  5. 打包生成未签名的.apk文件
  6. 对未签名.apk文件进行签名
  7. 对签名后的.apk文件进行对齐处理

apk打包工具介绍

工具名称 功能介绍
aapt Android资源打包工具
aidl Android接口描述语言转化为.java文件的工具
javac Java Compiler,java代码转class文件
dex 转化.class文件为Davik VM能识别的.dex文件
apkbuilder 生成apk包
jarsigner .jar文件的签名工具
zipalign 字节码对齐工具
